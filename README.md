# Simple Calculator 🧮

This is a basic calculator project built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input handling
- Error handling (division by zero)

## Technologies Used
- Python
- Google Colab

## How to Run
1. Open the file in Google Colab or any Python IDE
2. Run the code
3. Enter numbers and operations
